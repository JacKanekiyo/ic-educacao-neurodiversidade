# Educação sob Múltiplas Dimensões: Análise de Perfis de Aprendizagem e Neurodiversidade no Brasil 

Este repositório contém o projeto de Iniciação Científica focado na análise do impacto de fatores socioeconômicos e de saúde (TEA e TDAH) no desempenho escolar brasileiro. Utilizando técnicas de **Ciência de Dados** e **Machine Learning**, o estudo integra múltiplas bases de dados governamentais para identificar padrões complexos na educação nacional.

##  Destaques Técnicos

* **Processamento de Big Data**: Implementação de técnicas de amostragem e redução para manipulação de grandes volumes de registros.
* **Pipeline de Dados Modular**: Estrutura organizada em etapas de pré-processamento, limpeza especializada e integração final.
* **Modelagem Estatística**: Uso de algoritmos de **Random Forest** e **Clusterização** para análise de perfis multidimensionais.

##  Tecnologias Utilizadas

* **Linguagem**: Python (Pandas, NumPy, Scikit-Learn).
* **Bases de Dados**: Microdados do SAEB (Inep), Censo Escolar e DATASUS.
* **Visualização**: Matplotlib, Seaborn e Power BI.

##  Principais Insights da Pesquisa

* **Fator Renda**: A análise revelou que a renda familiar é o preditor mais forte para o desempenho escolar, superando impactos isolados de diagnósticos.
* **Barreira de Diagnóstico**: Identificação de subnotificação significativa em categorias de saúde por limitações de acesso.
* **Disparidade de Gênero**: Observou-se heterogeneidade na prevalência de condições de neurodesenvolvimento entre os sexos.
