**Educação sob Múltiplas Dimensões: Uma Análise de Perfis de Aprendizagem e Neurodiversidade no Brasil 
Este repositório contém o projeto de Iniciação Científica focado na análise do impacto de fatores socioeconômicos e de saúde (TEA e TDAH) no desempenho escolar brasileiro. Utilizando técnicas avançadas de Ciência de Dados e Machine Learning, o estudo integra múltiplas bases de dados governamentais para identificar padrões complexos na educação nacional.**

 **Destaques Técnicos**
Processamento de Big Data: Implementação de técnicas de amostragem e redução de dados para manipulação eficiente de grandes volumes de registros do governo.

Pipeline de Dados Modular: Estrutura organizada em etapas claras de pré-processamento, limpeza especializada por nível de ensino e integração final de bases heterogéneas.

Modelagem Estatística: Uso de algoritmos de Random Forest e Clusterização para análise de perfis multidimensionais.

 **Tecnologias Utilizadas**
Linguagem: Python (Pandas, NumPy, Scikit-Learn).

Bases de Dados: Microdados do SAEB (Inep), Censo Escolar e DATASUS.

Visualização: Matplotlib, Seaborn e Power BI.

**Principais Insights da Pesquisa**
Fator Renda: A análise de Feature Importance revelou que a renda familiar é o preditor mais forte para o desempenho em Matemática, superando o impacto isolado de diagnósticos neurodivergentes.

Barreira de Diagnóstico: Identificação de subnotificação significativa em categorias de saúde devido a limitações estruturais de acesso, impactando o perfil multidimensional dos alunos.

Disparidade de Gênero: Observou-se heterogeneidade na prevalência de TEA e TDAH entre sexos, influenciando as trajetórias de aprendizagem.
